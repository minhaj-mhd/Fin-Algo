## 📝 Session Summary

### Context
User asked to run a fresh backtest on data the 1-hour model was never trained on, to verify the insane sniper strategy results from the Complete Edge Catalog (68–76% win rates). Jupyter MCP was unavailable so backtest was run as a Python script directly.

### Key Technical Discovery: The IBS Lookahead Effect
The research calibrated thresholds on the **14:30 bar** (the last 1-hour bar of the day). This bar's features include `IBS = (Close − Low) / (High − Low)`, which uses the bar's own close price — only known at **15:30 = market close**. The research's "win" check also measured the same bar's return. This creates an intrabar lookahead: the model was scored on a completed bar and validated against that same completed bar's performance.

In live trading, you can only trade AFTER a bar closes:
- 14:30 bar closes at 15:30 = market closed, can't enter.
- **13:30 bar closes at 14:30 = tradeable entry.** This is the correct signal source.

### The Gap Explained
| Tier | Research WR | Tradeable WR | Gap | Cause |
|---|---|---|---|---|
| A | 74–76% | 57.9% | −18% | IBS lookahead + intrabar Return feature |
| B | 68% | 54.8% | −13% | Same |
| D | 60% | 60.0% | 0% | Dual-lock structure self-corrects |

### Tier C Structural Finding
The `long_score` minimum at the 13:30 bar in the entire Jan–May 2026 period was −0.1668 — just barely above the −0.167 threshold. **Zero Tier C signals fire from the 13:30 bar.** The extreme negative long conviction signal is a last-hour-only phenomenon that the model only generates when it has seen the completed 14:30 bar features. Tier C is inaccessible with a 1-bar lag.

## 📝 Session Extension — Research vs Backtest Gap Diagnosis

User asked: "did you check the images — was the research wrong or your backtest?"

### Root Cause: IBS Evaluation Leakage
The research's 68–76% WR numbers were computed on **completed 14:30 bar features** measured against **that same bar's return**. Both the signal (IBS = (Close-Low)/(High-Low)) and the outcome check (Return < 0) require the 14:30 bar's 15:30 close price. This is evaluation lookahead — not training leakage.

The model itself is clean. The win rate inflation came entirely from the evaluation methodology using future-within-bar information as both input and output.
